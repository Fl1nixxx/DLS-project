# DLS-project

### Проект по компьютерному зрению для подсчета площади застройки по снимку с воздуха

Реализовано:
- Архитектура модели
- Training loop
- Инференс модели (streamlit приложение)
- Подсчёт площади построект

Для запуска сайта с приложением необходимо скачать файл **launch_app.ipynb** и выполнить приложенный код<br>
Модель работает с данными в формате **.tif**, **.tfif**

> [!NOTE]
> В данный момент проект полностью функционирует , производится улучшение качества модели
