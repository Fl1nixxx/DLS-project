# DLS-project

### Проект по компьютерному зрению для подсчета площади застройки по снимку с воздуха

Реализовано:
- Архитектура модели
- Training loop
- Инференс модели (streamlit приложение)

Для запуска приложения необходимо скачать файл **launch_app.ipynb** и выполнить приложенный код<br>
Формат данных в приложении: **.tfif**, **.tif**
