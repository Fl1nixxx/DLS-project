# DLS-project

### Проект по компьютерному зрению для подсчета площади застройки по снимку с воздуха

Реализовано:
- Архитектура модели
- Training loop
- Инференс модели (Streamlit-приложение)
- Подсчёт площади застроек

> [!IMPORTANT]
> **Модель работает с данными в формате .tif, .tiff**

> [!NOTE]
> **Веса модели:** Скачать обученные веса (файл `.pth`) можно в разделе **[Releases](https://github.com)** этого репозитория.

### Запуск приложения
Для запуска сайта с приложением необходимо скачать файл **launch_app.ipynb** и выполнить приложенный код.

> [!NOTE]
> Если сайт с приложением не открывается, немного подождите и зайдите по ссылке еще раз.
