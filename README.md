# DLS-project

### Проект по компьютерному зрению для подсчета площади застройки по снимку с воздуха

Реализовано:
- Архитектура модели
- Training loop
- Инференс модели (streamlit приложение)

Для запуска приложения необходимо скачать файл **launch_app.ipynb** и выполнить приложенный код. 
Если сайт с приложением не открывается , немного подождите и зайдите по ссылке еще раз
