# DLS-project

### Проект по компьютерному зрению для подсчета площади застройки по снимку с воздуха

--- 

Реализовано:
- Модель и обучение: спроектирована архитектуруна и реализован процесс обучения
- Инференс: запущено рабочее веб-приложение на Streamlit.
- Подсчет площади: добавлена функция автоматического расчета площади застройки.

> [!IMPORTANT]
> **Модель работает с данными в формате .tif, .tiff .png .jpeg**

---

### Модель и архитектура

В качестве базовой модели выбрана **U-Net++** с измененными сверточными слоями. Также в сеть интегрирован механизм Squeeze-and-Excitation (SE).  
Энкодером модели служит предобученный **resnet50**  

---

### Обучение

**Loss**

В обучении модели использована комбинация **BCE** + **Dice** + **Boundary** loss

**BCE** - правильная классификация каждого пикселя
**Dice** - оптимизирует iou метрику
**Boundary** - Точность контуров и краев зданий

**Остальные параметры**
- **optimizer** : AdamW
- **lr** : 0.003
- **scheduler** : ReduceLROnPlatea
- **Batch size** : 16
- **epochs** : 50

---

### Подсчет площади

Подсчёт площади

Подсчёт площади застройки реализован двумя способами:

**1. Автоматическое определение масштаба**

Если загруженное изображение имеет формат .tif или .tiff и содержит геопространственные метаданные GeoTIFF, приложение автоматически определяет пространственное разрешение снимка в метрах на пиксель.

>[!NOTE]
>Обычный TIFF-файл может не содержать геопривязку. В этом случае масштаб необходимо указать вручную.

**2. Ручной ввод масштаба**

Для изображений в форматах .png, .jpg и .jpeg, а также для TIFF-файлов без геопространственных метаданных пользователь должен самостоятельно указать масштаб изображения в метрах на пиксель.

>[!IMPORTANT]
>Для корректного расчёта площади у изображения должен быть известен масштаб.

### Результаты и качество модели

* **Среднее IoU (Intersection over Union):** `0.76` на валидационной выборке.

**Известные ограничения и проблемы сегментации:**
* **Геометрия замкнутых контуров:** наблюдаются дефекты распознавания и ошибки подсчета площади в домах кольцевой/замкнутой архитектуры (круглые дома, дома-колодцы). Модель может неверно определять внутренние дворовые пространства или разрывать контур.
* **Текстурные артефакты:** Модель может распознавать дорогу или участок растительности как дом, либо же не полносью выделять строение.
