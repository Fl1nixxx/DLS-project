# DLS-project

### Проект по компьютерному зрению для подсчета площади застройки по снимку с воздуха

--- 

Реализовано:
- Модель и обучение: спроектирована архитектуруна и реализован процесс обучения
- Инференс: запущено рабочее веб-приложение на Streamlit.
- Подсчет площади: добавлена функция автоматического расчета площади застройки.

> [!IMPORTANT]
> **Модель работает с данными в формате .tif, .tiff .png .jpeg**

---

### Модель и архитектура

В качестве базовой модели выбрана **U-Net++** с измененными сверточными слоями. Также в сеть интегрирован механизм Squeeze-and-Excitation (SE).  
Энкодером модели служит предобученный **resnet50**  

---

### Результаты и качество модели

* **Среднее IoU (Intersection over Union):** `0.75` на валидационной выборке.

**Известные ограничения и проблемы сегментации:**
* **Геометрия замкнутых контуров:** наблюдаются дефекты распознавания и ошибки подсчета площади в домах кольцевой/замкнутой архитектуры (круглые дома, дома-колодцы). Модель может неверно определять внутренние дворовые пространства или разрывать контур.
* **Текстурные артефакты:** Модель может распознавать дорогу или участок растительности как дом, либо же не полносью выделять строение.
